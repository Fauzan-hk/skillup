## 🚀 Tentang Proyek Ini
Proyek ini dibuat untuk menampilkan antarmuka yang tidak hanya fungsional tetapi juga sangat menarik secara visual. Dengan inspirasi dari gaya anime dan efek partikel, setiap baris kode ditulis dengan presisi.

## ✨ Fitur Utama
- **Estetika Visual:** Menggabungkan gambar animasi (GIF) untuk mensimulasikan efek partikel.
- **Desain Responsif:** Komponen antarmuka dirancang agar berfungsi di berbagai ukuran layar.
- **Performa Tinggi:** Dioptimalkan untuk memuat lebih cepat tanpa mengorbankan kualitas visual.

## 📌 Catatan Developer
> **Info:** File Markdown (`.md`) secara bawaan tidak mendukung kode CSS atau JavaScript yang beroperasi langsung untuk merender partikel web interaktif seperti `particles.js`. Oleh karena itu, cara "paling keren" dan didukung secara universal di GitHub/Markdown adalah menggunakan gambar animasi berformat `.gif` (seperti yang digunakan di atas) yang sudah memiliki efek animasi partikel.
